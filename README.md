# 零基础搭建自己的龙虾

面向新手的 OpenClaw 搭建与学习资料仓库。

目标：把分散在官方文档、社区教程、视频和实战文章里的关键信息集中起来，形成一条可执行的上手路径。

## 你可以从这里开始

1. 先看 `docs/00-快速开始.md`（30 分钟跑通最小可用版本）
2. 再看 `docs/01-官方文档索引.md`（官方权威入口）
3. 按需查 `docs/02-社区教程索引.md`（不同平台和场景）
4. 视频学习看 `docs/03-视频教程索引.md`
5. 深入阅读看 `docs/04-文章教程索引.md`

## 仓库结构

- `docs/00-快速开始.md`：最短落地路径
- `docs/01-官方文档索引.md`：官方文档导航
- `docs/02-社区教程索引.md`：GitHub 社区教程与项目
- `docs/03-视频教程索引.md`：公开视频教程索引
- `docs/04-文章教程索引.md`：文章类教程索引
- `docs/05-搭建路线图.md`：从入门到稳定运行的路线图
- `docs/99-资料来源.md`：本仓库收录来源

## 使用原则

- 官方优先：涉及安装参数、安全配置、更新命令时，以官方文档为准。
- 社区补充：社区教程用于理解思路和不同部署场景，不直接照抄到生产环境。
- 安全先行：先最小权限、再逐步开能力。

## 免责声明

- 第三方教程可能过期、命令可能变化。
- 任何涉及密钥、远程访问、自动执行命令的步骤，请先在测试环境验证。

# 00 快速开始（新手最短路径）

## 目标

30 分钟内完成：

- OpenClaw 安装
- 初次 onboarding
- 跑通一次对话

## 前置条件

- Node.js 22+
- 一台可长期在线的机器（macOS / Linux / Windows + WSL2）
- 至少一个模型提供方凭证（如 Anthropic / OpenAI）

## 步骤

### 1) 安装 OpenClaw

```bash
npm install -g openclaw@latest
```

### 2) 运行引导流程

```bash
openclaw onboard --install-daemon
```

### 3) 检查状态

```bash
openclaw status
openclaw gateway status
```

### 4) 进行一次本地测试

```bash
openclaw agent --message "你好，做个 3 行自我介绍" --thinking low
```

## 建议默认配置（给新手）

- 先只接一个渠道（如 Telegram）
- 先不开高风险技能
- 先只允许自己访问（allowlist）
- 把自动更新和备份计划写入日历

## 常见卡点

- 命令不存在：检查 Node 版本与全局 PATH
- Gateway 启动失败：先看 `openclaw doctor`
- 渠道消息不回：先看 token、权限、allowlist
